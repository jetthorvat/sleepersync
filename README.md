# SleeperSync

Local-first fantasy football draft dashboard for Sleeper.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS + shadcn/ui
- TanStack Query
- Dexie (IndexedDB)
- Zustand
